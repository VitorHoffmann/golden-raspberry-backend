Para rodar o projeto basta digitar

mvn spring-boot:run

no terminal/power shell do windows apontado para o projeto

A aplicação estará rodando no endereço: http://localhost:8080

Endpoint para testar a funcionalidade: http://localhost:8080/api/movies

Para rodar os testes do projeto basta digitar

mvn test

no terminal/power shell do windows apontado para o projeto
